# 🩺 Diabetes Prediction using Machine Learning

## 📌 Overview
This project predicts whether a person has diabetes using Machine Learning. It is built using Python and the Scikit-learn library. The model is trained on the Pima Indians Diabetes Dataset and predicts whether a patient is diabetic based on medical attributes.

## 🚀 Technologies Used
- Python
- Google Colab
- NumPy
- Pandas
- Matplotlib
- Scikit-learn

## 📂 Dataset Features
- Pregnancies
- Glucose
- Blood Pressure
- Skin Thickness
- Insulin
- BMI
- Diabetes Pedigree Function
- Age
- Outcome (Target)

## ⚙️ Workflow
1. Import required libraries
2. Load the dataset
3. Explore the dataset
4. Split the dataset into training and testing sets
5. Train the Support Vector Machine (SVM) model
6. Evaluate the model using accuracy score
7. Predict whether a patient has diabetes

## 🤖 Machine Learning Algorithm
**Support Vector Machine (SVM)**

## 📊 Result
The trained SVM model predicts whether a patient is diabetic or non-diabetic based on the given medical information.
